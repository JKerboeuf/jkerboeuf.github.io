## Description

Une simple application Winforms réalisée comme un travail scolaire.  
Le sujet m'a chargé de réaliser un logiciel pour la médiathèque imaginaire "Mediatek86" permettant de gérer les salariés et leurs absences à l'aide d'une base de données SQL.

## Fonctionnalités

- Utilise une connexion sécurisée à la base de données pour accéder au logiciel
- Afficher, ajouter, modifier et supprimer des employés de la base de données
- Afficher, ajouter, modifier et supprimer les absences des employés
- Inclus un installateur
- Inclus une documentation technique complète

## Descriptif technique

Le logiciel a été réalisé avec **.NET Framework WinForms** en **C#** sur l'IDE **Visual Studio** dans une structure MVC, il exploite une base de données **MySQL**.  
La connexion initiale est cryptée avec SHA256 et est résistante aux injections SQL.

## Contexte

La médiathèque Mediatek86 a besoin d'un outil facilitant la gestion de son personnel que le gestionnaire utilisera.
Mediatek86 demande un logiciel permettant au responsable du personnel de gérer facilement le personnel.
Le logiciel permet de lister le personnel et leurs absences, il permet également d'ajouter, de modifier ou de supprimer du personnel ou des absences.
